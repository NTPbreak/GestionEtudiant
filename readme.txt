Projet : Gestion des étudiants et des notes

Ce projet consiste à créer une application Java permettant de gérer une liste d'étudiants, leurs matières et leurs notes. L'application fournit des fonctionnalités telles que le calcul des moyennes, le classement des étudiants par ordre de mérite, et la visualisation des étudiants ayant une moyenne supérieure ou égale à la moyenne de la classe.


## Prérequis

Avant de lancer ce projet, assurez-vous d'avoir les éléments suivants installés sur votre machine :

1. Java Development Kit (JDK) : Version 8 ou ultérieure (le 23 a été utiliser ici).


2. IDE Java : Un environnement de développement intégré tel que IntelliJ IDEA, Eclipse ou NetBeans.

- IntelliJ IDEA

- Eclipse

- NetBeans

- Visual studio code


3. Git : Pour cloner le dépôt du projet.

Lien de téléchargement de Git: https://github.com/NTPbreak/GestionEtudiant.git


## Installation


1. Importez le projet dans votre IDE Java :

	- Ouvrez votre IDE.

	- Sélectionnez "Importer un projet" ou "Ouvrir un projet existant".

	- Naviguez jusqu'au dossier contenant le projet cloné et sélectionnez-le.

2. Configurez le projet :

	- Assurez-vous que le JDK est configuré dans votre IDE et que le path aussi.

	- Vérifiez que les dépendances (le cas échéant) sont correctement configurées.

## Utilisation

1. Compilez le projet.

	- Utilisez la commande "Build" dans votre IDE ou exécutez la commande suivante dans le terminal :
	javac Examen/*.java Inscription/*.java
2. Lancez le programme principal :

	- Utilisez la commande "Run" dans votre IDE ou exécutez la commande suivante dans le terminal :
	java Inscription/GestionDesEtudiants
